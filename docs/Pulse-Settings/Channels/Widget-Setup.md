# Widget Setup

The Widget Setup tab controls the appearance, behavior, branding, and installation of the website chat widget.

These settings determine how visitors experience the chatbot on your website.

---

# Appearance

### Brand Logo

Upload your organization's logo.

#### Supported Formats

- PNG
- JPG
- SVG

### Primary Color

Used for:

- Header
- Buttons
- Interactive elements

### Secondary Color

Used for:

- Secondary actions
- Accent elements

### Chat Bubble Color

Color of visitor chat messages.

### Agent Message Color

Color of chatbot or agent messages.

### Background Color

Defines the chat window background.

### Font Family

Supported options include:

- Default
- Inter
- Roboto
- DM Sans
- System Font

### Font Size

Options:

- Small
- Medium
- Large

---

## Widget Launcher (Chat Bubble)

Controls the chat icon displayed on the website.

### Launcher Type

Options include:

- Chat Bubble
- Floating Button
- Text Button
- Custom Icon

### Custom Image Upload

Allows administrators to upload a custom launcher icon.

### Launcher Text

Examples:

- Chat with Us
- Need Help?
- Support

### Bubble Size

Options:

- Small
- Medium
- Large

---

## Widget Position

Controls where the chat widget appears.

### Available Positions

- Bottom Right
- Bottom Left

### Right Margin

Distance from the right edge of the screen.

### Bottom Margin

Distance from the bottom edge of the screen.

### Z-Index

Controls display priority when overlapping with website elements.

---

## Chat Window Layout

Defines how the chat window is displayed.

### Chat Window Title

Example:

> Support Chat

### Subtitle

Example:

> We typically reply in a few minutes.

### Header Logo

Optional logo displayed within the chat window.

### Window Size

Options:

- Compact
- Medium
- Large

### Window Style

Options:

- Rounded
- Minimal
- Card Style

### Chat Avatar

Profile image displayed for chatbot messages.

---

## Chat Widget Behavior

### Auto Open Chat Window

Automatically opens the chat window after a specified delay.

### Auto Show Bubble

Displays the launcher bubble automatically after a configured time.

### Show Only After Scroll

Displays the widget after visitors scroll for a specified percentage of the page.

### Show on Exit Intent

Displays the widget when the visitor appears ready to leave the website.

---

## Welcome & Messages

### Welcome Message

Displayed when the chat window first opens.

Example:

> Hello! How can we help you today?

---

## Pre-Chat Form

Collects visitor information before a conversation begins.

### Enable Pre-Chat Form

When enabled, visitors may be asked to provide:

- Name
- Email
- Phone Number
- Company
- Custom Questions

:::note
This information can be used for lead generation and qualification.
:::

---

## Engagement Triggers

Used to proactively engage website visitors.

### Proactive Message

Example:

> Need help finding something?

#### Trigger Conditions

Messages can be triggered based on:

- Time spent on page
- Scroll percentage
- Number of pages visited
- Returning visitor status

---

## Mobile Experience

Controls chatbot behavior on mobile devices.

### Mobile Widget Enabled

Enable or disable the chatbot on mobile devices.

### Mobile Window Style

Options:

- Full Screen
- Floating Chat

### Auto Open on Mobile

Automatically opens the chatbot for mobile visitors.

---

## Sound & Notifications

### New Message Sound

Plays a sound when a new message is received.

#### Sound Types

- Ping
- Chime
- None

### Browser Notifications

Displays browser-based message notifications.

---

## Advanced Settings

### Persistent Chat

Preserves conversations across page refreshes.

### Visitor Tracking Enabled

Tracks visitor interactions across sessions.

---

### URL Rules

Control where the chatbot appears.

Examples:

- Show on all pages
- Show only on specific pages
- Hide on specific pages

---

## Installation

### Installation Script

Administrators can copy the generated script and add it to their website.

### Domain Restriction

Restrict widget usage to approved domains.

Examples:

- company.com
- support.company.com
